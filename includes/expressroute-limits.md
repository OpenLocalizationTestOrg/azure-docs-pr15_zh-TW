#### <a name="expressroute-limits"></a>ExpressRoute 限制

下列限制會套用至每個訂閱的 ExpressRoute 資源。

| 資源 | 預設的限制 |
|---|---|
| 每個訂閱 ExpressRoute 電路 | 10 |
| 每個地區每個訂閱的手錶 ExpressRoute 電路 | 10 |
| 最大的數字的 ExpressRoute 標準 Azure 私人對等路由 | 4000 |
| 路由具備 ExpressRoute 進階版增益集 Azure 私人對等的最大數目 | 10000 |
| 最大的數字的 ExpressRoute 標準 Azure 公用對等路由 | 200 |
| 路由具備 ExpressRoute 進階版增益集 Azure 公用對等的最大數目 | 200 |
| 最大的數字的 Azure Microsoft 對等的 ExpressRoute 標準路由 | 200 |
| 最大的數字的 Azure ExpressRoute 進階版附加元件的對等的 Microsoft 路由 | 200 |
| 每個 ExpressRoute 電路所允許的虛擬網路連結數目 | 請參閱下表 |

| 100 Mbps | 10 | 25 |
| 200 Mbps | 10 | 25 |
| 500 Mbps | 10 | 40 |
| 1 Gbps | 10 | 50 |
