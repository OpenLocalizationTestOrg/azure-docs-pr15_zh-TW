## <a name="step-1---start-your-script"></a>步驟 1-啟動您的指令碼

您可以下載用的完整艦隊指令碼[以下](https://raw.githubusercontent.com/Azure/azure-quickstart-templates/master/IaaS-Story/03-Static-public-IP/virtual-network-deploy-static-pip-arm-cli.sh)。 請遵循下列步驟來變更搭配使用您的環境中的指令碼。

1. 變更您想要使用您的部署中的數值為根據的變數下列的值。 此文件中使用此案例地圖以下的值。

        # Set variables for the new resource group
        rgName="IaaSStory"
        location="westus"
        
        # Set variables for VNet
        vnetName="TestVNet"
        vnetPrefix="192.168.0.0/16"
        subnetName="FrontEnd"
        subnetPrefix="192.168.1.0/24"
        
        # Set variables for storage
        stdStorageAccountName="iaasstorystorage"
        
        # Set variables for VM
        vmSize="Standard_A1"
        diskSize=127
        publisher="Canonical"
        offer="UbuntuServer"
        sku="14.04.2-LTS"
        version="latest"
        vmName="WEB1"
        osDiskName="osdisk"
        nicName="NICWEB1"
        privateIPAddress="192.168.1.101"
        username='adminuser'
        password='adminP@ssw0rd'
        pipName="PIPWEB1"
        dnsName="iaasstoryws1"

## <a name="step-2---create-the-necessary-resources-for-your-vm"></a>步驟 2-您 VM 建立所需的資源

建立 VM 之前, 必須 VM 所使用的資源群組、 VNet、 公用 IP，與 NIC。

1. 建立新的資源群組。

        azure group create $rgName $location
        
2. 建立 VNet 和子網路。
        
        azure network vnet create --resource-group $rgName \
            --name $vnetName \
            --address-prefixes $vnetPrefix \
            --location $location
        azure network vnet subnet create --resource-group $rgName \
            --vnet-name $vnetName \
            --name $subnetName \
            --address-prefix $subnetPrefix

3. 建立公用 IP 資源。 

        azure network public-ip create --resource-group $rgName \
            --name $pipName \
            --location $location \
            --allocation-method Static \
            --domain-name-label $dnsName 

4. 建立網路介面 (NIC) 建立上述的公用 IP 子網路中 vm。 請注意命令的第一組用來擷取子網路先前所建立的**識別碼**。

        subnetId="$(azure network vnet subnet show --resource-group $rgName \
                        --vnet-name $vnetName \
                        --name $subnetName|grep Id)"

        subnetId=${subnetId#*/}
        
        azure network nic create --name $nicName \
            --resource-group $rgName \
            --location $location \
            --private-ip-address $privateIPAddress \
            --subnet-id $subnetId \
            --public-ip-name $pipName

    >[AZURE.TIP] 上述的第一個命令會使用[grep](http://tldp.org/LDP/Bash-Beginners-Guide/html/sect_04_02.html)和[字串操作](http://tldp.org/LDP/abs/html/string-manipulation.html)（更明確地說，子字串移除）。 

5. 建立儲存帳戶裝載 VM OS 磁碟機。

        azure storage account create $stdStorageAccountName \
            --resource-group $rgName \
            --location $location --type LRS 

## <a name="step-3---create-the-vm"></a>步驟 3-建立 VM 

現在，所有必要的資源是在 [位置]，您可以建立新的 VM。

1. 建立 VM。

        azure vm create --resource-group $rgName \
            --name $vmName \
            --location $location \
            --vm-size $vmSize \
            --subnet-id $subnetId \
            --nic-names $nicName \
            --os-type linux \
            --image-urn $publisher:$offer:$sku:$version \
            --storage-account-name $stdStorageAccountName \
            --storage-account-container-name vhds \
            --os-disk-vhd $osDiskName.vhd \
            --admin-username $username \
            --admin-password $password

2. 儲存的指令碼檔。

        sh myscript.sh
