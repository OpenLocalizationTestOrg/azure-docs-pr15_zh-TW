### <a name="august-26-2016"></a>2016 年 8 月 26

Azure 工具組蝕-2016 年 8 月發行的包含下列增強功能︰

* **自訂 JDK 散佈**。 Azure 工具組的蝕現在支援指定及部署 Azure WebApp 容器的任意 JDK 版本︰
  - 除了提供 Azure JDKs，您也可以選擇從寬祖魯文 OpenJDK 版本可供 Azure Azul 系統的選項。
  - 如果您上傳其為 ZIP 檔案至您儲存的帳戶，您也可以指定您自己的 JDK 通訊群組。
* **Azure 總管檢視的增強功能**︰
  - 使用新資源管理員模型的 Azure 虛擬機器管理支援︰ 您可以清單、 建立及刪除資源管理員為基礎的虛擬機器，而不需離開 IDE。

Azure 外掛程式-12 月 2011 CTP 蝕已發行。 新的功能包括︰

* **工作階段相關性 (&quot;自黏工作階段&quot;) 支援︰**協助啟用狀態，群組直條圖 Java 應用程式，只要的單一的核取方塊。 如需詳細資訊，請參閱[工作階段相關性]。
* **預先製作的啟動指令碼範例︰**最常用 Java 伺服器 （Tomcat Jetty、 JBoss、 GlassFish），讓您可以只複製/貼上來自您的專案，請在啟動指令碼範例目錄。
* **模擬器啟動輸出即時︰**您現在可以觀看執行所有步驟從啟動指令碼視窗中的專用的主控台顯示您的進度和失敗指令碼為 Azure 執行它。
* **自動的精簡 java.exe 監控︰**Java.exe 停止執行中，使用您的部署中自動包含輕量型、 預先製作指令碼時的會強制角色資源。
* **偵錯時，設定使用者介面的遠端 Java 應用程式︰**可讓您輕鬆地啟用蝕的存取 Java 應用程式，讓您可以逐步執行並偵錯即時 Java 程式碼，在模擬器或 Azure 雲端中，執行的遠端偵錯工具。 如需詳細資訊，請參閱[在蝕偵錯 Azure 應用程式]。
* **本機存放區的資源設定 UI:**因此您不再需要設定本機資源直接處理 XML。 此功能也可讓您透過環境變數，您可以直接從您啟動指令碼參考部署後存取您的本機資源的有效的檔案路徑。 如需詳細資訊，請參閱[本機存放區屬性]。
* **環境變數設定 UI:**因此您不再需要設定環境變數，透過設定 XML 手動編輯。 如需詳細資訊，請參閱[環境變數屬性]。
* **SQL Azure JDBC 驅動程式︰**取得安裝外掛程式透過為完美整合式蝕文件庫，以便更容易程式設計 SQL Azure。 
* **快速的操作功能表存取角色設定 UI**︰ 只要以滑鼠右鍵按一下 [角色] 資料夾中，然後按一下 [**內容**。
* **自訂 Azure 專案和角色資料夾圖示︰**增加可見性和您的工作區和專案的更輕鬆導覽。
