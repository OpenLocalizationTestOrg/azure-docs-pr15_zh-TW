# <a name="storsimple-snapshot-manager-user-interface"></a>StorSimple 快照管理員使用者介面

## <a name="overview"></a>概觀

StorSimple 快照管理員的您可以使用並管理備份直覺化的使用者介面。 本教學課程介紹使用者介面，，然後再說明如何使用每個元件。 [StorSimple 快照管理員的詳細說明，請參閱[什麼是 StorSimple 快照管理員？](storsimple-what-is-snapshot-manager.md)

![StorSimple 快照管理員窗格](./media/storsimple-use-snapshot-manager/HCS_SSM_gui_panes.png)

主控台視窗有五個主要的項目。 按一下適當連結的完整描述，每個項目。

- [功能表列](#menu-bar)

- [範圍] 窗格](#scope-pane)

### <a name="console-accessibility"></a>主控台協助工具

StorSimple 快照管理員使用者介面支援協助工具功能提供的 Windows 作業系統和 Microsoft 管理主控台 (MMC)，以及一些 StorSimple 快照管理員特定的鍵盤快速鍵。 

- Windows 的協助工具功能的說明，請移至[視窗的鍵盤快速鍵](https://support.microsoft.com/kb/126449)。 

- 如需 MMC 協助工具功能的說明，請移至[MMC 3.0 的協助工具](https://technet.microsoft.com/library/cc766075.aspx)

- StorSimple 快照管理員的協助工具功能的說明，請移至[鍵盤快速鍵和快速鍵](#keyboard-navigation-and-shortcuts)。
