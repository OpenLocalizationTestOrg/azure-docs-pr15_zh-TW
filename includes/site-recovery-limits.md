下列限制會套用至 Azure 網站復原︰


|限制識別碼|預設的限制|
|---|---|---|---|
|數字的每個訂閱的保存庫|25|
|每個 Azure 保存庫伺服器的數目|250|
|數字的每個 Azure 保存庫保護群組|沒有限制|
|數字的每個 Azure 修復計劃地窖|沒有限制|
|每個保護群組伺服器的數目|沒有限制|
|每個復原計劃的伺服器的數目|50|
