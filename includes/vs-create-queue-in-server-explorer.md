您可以使用 Visual Studio**伺服器總管**建立 Azure 佇列。

![伺服器 Explorer 二進位大型物件][Image1]

1. 在 [**檢視**] 功能表中，選擇 [**伺服器總管**]。
2. 在伺服器總管] 中，展開您的訂閱的**Azure**節點，展開的**儲存空間**節點和您指定連線的 Azure 儲存體服務中的儲存空間帳戶的節點。
3. 選取 [**佇列**] 節點，然後從操作功能表中選擇 [**建立佇列**。
4. 輸入佇列中的名稱，然後選擇**[確定**]。
