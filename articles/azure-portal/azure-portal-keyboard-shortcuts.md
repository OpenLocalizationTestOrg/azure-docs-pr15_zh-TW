# <a name="azure-portal-keyboard-shortcuts"></a>Azure 入口網站的鍵盤快速鍵

## <a name="open-hubs"></a>開啟集線器

| 快速鍵 | 巨集指令 |
|--------|----------|
| N | 通知 |
| / | 瀏覽服務 |
| A | 使用中工作階段 |
| B | 帳單 |
| C | 建立 |

## <a name="navigation"></a>瀏覽

| 快速鍵 | 巨集指令 |
|--------|----------|
| H | Azure 開始儀表板 |
| F | 第一開啟把 |
| J | 先前開啟的刀 |
| K | 下一步開啟刀 |
| L | 最後一個開啟刀 |
| - | 刀最小化或還原從最大化 |
| + | 最大化刀或還原從最小化 |
| # | 關閉目前刀 |
| ¦ | 關閉所有刀 |
| , | 將焦點移到命令列 |
| . | 將焦點移至導覽列 |
| ? | 檢視鍵盤快速鍵 |
