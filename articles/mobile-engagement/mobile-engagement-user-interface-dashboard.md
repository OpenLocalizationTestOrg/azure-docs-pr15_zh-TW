# <a name="how-to-view-and-manage-charts-of-data-about-your-application"></a>如何檢視及管理應用程式的相關資料的圖表
儀表板區段 UI 的可讓客戶建立自訂的圖表，最大化提供所需的資訊，而不是在 [分析] 區段中搜尋他們想要的時間。 儀表板顯示的趨勢的應用程式和作用中使用者計數不同版本的應用程式在給定的時間 （小時/日/週/月/自訂）。 顯示趨勢依據前 7 天。

### <a name="see-also"></a>另請參閱
-  [疑難排解指南-狀況分析][Link 21]

## <a name="dashboard"></a>儀表板
- 您可以新增圖表至空白的儀表板任何圖表，您想要在儀表板上看到的使用者介面的 [分析] 區段中的齒輪功能表中選擇的 「 傳送至 [儀表板 」 的動作。
 
## <a name="-add-a-dashboard"></a>+ （新增儀表板）
- 若要建立新的儀表板，會顯示在功能區] 功能表上，您可以使用 [+] 功能區功能表項目。
 
## <a name="gear-menu"></a>齒輪功能表
- 您可以使用 [在儀表板] 齒輪功能表刪除、 重新命名共用]，或您的儀表板的排程 （電子郵件接收）。 您可以使用 [排程] 選項，以定義您想要透過電子郵件收到您自訂的儀表板的排程。 您可以排程收到每日、 週或月。 您可以選擇的日期及一天的週或當您想要接收電子郵件中的月份天的時間。 您的電子郵件地址和時區會根據您設定的使用者介面的 [我的帳戶] 區段中的設定。
