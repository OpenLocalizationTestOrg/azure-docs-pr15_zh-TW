# <a name="azure-ad-and-applications-assigning-users-to-an-application"></a>Azure AD 和應用程式︰ 將使用者指派至應用程式
您可以將使用者和群組指派的應用程式之前，您必須要求使用者指派。  若要瞭解如何要求使用者指派，請參閱[要求使用者指派](active-directory-applications-guiding-developers-requiring-user-assignment.md)一文。

## <a name="assigning-users-to-an-application"></a>將使用者指派至應用程式
1. Azure 入口網站以系統管理員帳戶登入。
2. 按一下 [**所有項目**中的項目主功能表上。
3. 選擇您使用應用程式的目錄。
4. 按一下 [**應用程式**] 索引標籤。
5. 從這個目錄相關聯的應用程式清單中選取的應用程式。
6. 按一下 [**使用者及群組**] 索引標籤。
8. 選取您想要指派的應用程式的使用者。
9. 按一下 [**指派**]。
10. 按一下 [ **]**出現提示時。
