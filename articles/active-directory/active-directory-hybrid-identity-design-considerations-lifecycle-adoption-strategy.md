# <a name="determine-hybrid-identity-lifecycle-adoption-strategy"></a>決定混合式身分識別生命週期採用策略
在此工作中，您會定義您的混合式身分識別解決方案，以符合您所定義[判斷混合式身分識別管理](active-directory-hybrid-identity-design-considerations-hybrid-id-management-tasks.md)工作的業務需求的身分識別管理策略。

## <a name="role-based-access-control"></a>角色型存取控制
角色型存取控制 (RBAC) 使用角色與佈建原則評估，測試，然後強制執行您的商務程序和規則授與存取權的使用者。 主要管理員建立佈建原則指派給角色的使用者，並的資源的下列角色定義的權利。 RBAC 延伸身分識別管理方案，使用軟體的程序，並減少提供的程序中的使用者手動互動。
Azure AD RBAC 可讓使用者以限制個別他有權存取 Azure 管理入口網站後，可以執行的作業的公司。 藉由使用 RBAC 來控制入口網站的存取權，IT 管理員 ca 委派存取權限使用下列存取管理方法︰

- **群組為基礎的角色指派**︰ 您可以從您的本機 Active Directory 可以同步處理的 Azure AD 群組指派存取。 這可讓您運用現有的投資的組織所做的工具和程序來管理群組。 您也可以使用 Azure AD 進階版的委派的群組管理功能。
- **使用內建於 Azure 中的角色**︰ 您可以使用下列三種角色，擁有人、 參與者及讀取程式，以確保使用者和群組有權限只需要完成工作的工作。
- **Granular 存取資源**︰ 您可以指派角色給使用者和群組用於特定訂閱、 資源] 群組中或個別的 Azure 資源，例如網站或資料庫。 如此一來，您可以確保使用者擁有其所需的所有資源的存取權和管理不需要的資源沒有存取。

## <a name="provisioning-and-other-customization-options"></a>佈建及其他自訂選項
您的小組可以使用哪些企業方案與需求，來決定多少自訂身分識別解決方案。 例如大型企業可能需要工作流程與從屬參照佈建廣泛用於地區的應用程式的時間列為基礎的自訂介面卡的階段的呈現計劃。 兩個或多個應用程式提供整個組織，測試成功後，可能會提供其他自訂計劃。 您可以自訂使用者應用程式的互動，並提供資源的程序可能會變更以容納自動化佈建。

您可以 deprovision 移除服務或元件。 例如，進行帳戶，表示刪除帳戶時，會從資源。

佈建資源的混合式部署模型結合要求與角色為基礎的方法，兩者都支援 Azure AD。 員工或受管理的系統的子集，商務可能會想要自動化存取角色為基礎的工作分派。 所有其他的存取要求或透過要求為基礎的模型的例外狀況，也可能會處理商務。 有些企業可能會以手動的工作分派的開始和趨近於混合式部署模型，以意圖完全角色部署在未來的發展。

其他公司可能會找到實用達成完成角色型佈建，商務原因，並目標需要目標的混合式方法。 仍其他公司可能滿意只要求基礎佈建，而不想要購買額外的工作，來定義並管理角色為基礎，自動化佈建原則。

## <a name="license-management"></a>管理授權
Azure AD 中的群組式授權管理可讓系統管理員將使用者指派至安全性群組和 Azure AD 自動將授權指派給群組的所有成員。 如果就新增，或從群組移除使用者時，授權將會自動指派或移除視中。

您可以使用您從同步處理的群組內部部署 AD 或管理 Azure AD 中。 此搭配 Azure AD 進階版自助群組管理您可以輕鬆地委派授權的指派，以適當的決策製造商。 您可以授權衝突等遺失位置資料問題會自動排序確保。

## <a name="define-synchronization-management"></a>定義管理的同步處理
將您的內部部署目錄整合與 Azure AD 可讓使用者更有效率存取內部部署和雲端資源提供常見的身分識別。 此整合使用者和組織可以利用下列任一動作︰

- 組織可在內部部署或雲端服務運用 Windows Server Active Directory，然後連線到 Azure Active Directory 提供使用者常見的混合式身分識別。
- 系統管理員可以提供條件存取權的應用程式資源、 裝置與使用者身分識別、 網路位置及多重因素驗證。
- Azure AD 至 Office 365，請 Intune SaaS 應用程式和協力廠商應用程式中，使用者可使用其常見的身分識別，透過帳戶。
- 開發人員可以建立的應用程式，運用常見的身分識別模型，到 Active Directory 內部部署或 Azure 整合應用程式，以雲端為基礎的應用程式
